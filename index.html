1. Difference between HTML tags and elements.

▶ HTML tags- Tags are the starting and ending part of an HTML element.
They begin with “<” symbol and end with “>”. Whatever written inside < and> are called tags.
For instance: -- 

<b></b>

▶ HTML elements – Elements enclose the contents in between the tags.
They consist of some kind of structure. It generally consists of a start tag, content and end tag.
For instance: --

<b>content</b>

2. Explain <br> tag?

▶ The <br> tag in HTML document is used to create a line break in a text.
It is an empty tag which means it does not need end tag like </br>.
It is generally used in poem and address where the division of line is necessary.

3. Explain how many different types of lists we have in HTML.

▶ In HTML we have THREE types of lists viz.
  1. Ordered List-
    In ordered list we have different types of lists
    • Numerical List (1,2,3, etc.)
    • Uppercase Letters (A,B,C, etc.)
    • Lowercase Letters (a,b,c, etc.)
    • Uppercase Roman Numerals (I,II,III, etc.)
    • Lowercase Numerals (i,ii,iii, etc.)
  2. Unordered List-
    In Unordered lists also we a different types
    • Circle type list
    • Disc type list
    • Square type list
    • None type list
  3. Description List -
    A description list is a list of terms, with a description of each term.

4. What are Sementic Tags?

▶ Sementic tags are nothing but tags that have meaning to them which means that tag clearly defines its content.
E.g <form>, <table> and <article>

5. Write a code to create an ordered list with roman numbers.

Code for Ordered List: 
        <ol type="i">
        <li>Mango</li>
        <li>Apple</li>
        <li>Banana</li>
        </ol>
Output:
  1. Mango
  2. Apple
  3. Banana

6. Write a code to create an unordered list with circle.

Code for Unordered List: 
        <ul type="circle">
        <li>Mango</li>
        <li>Apple</li>
        <li>Banana</li>
        </ul>
Output:
  • Mango
  • Apple
  • Banana
